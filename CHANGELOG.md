## [1.0.0] - 2024-06-28

### ✨ 新增功能

- 🚦 **红绿灯核心功能**
  - 四种状态显示：🟡 思考中、🟢 输出中、🔴 需要确认、⚪ 空闲
  - 实时状态更新（WebSocket 通信）
  - 平滑的颜色过渡动画

- 🖥️ **系统托盘集成**
  - 动态生成的红绿灯托盘图标
  - 右键菜单（显示/隐藏、退出）
  - 双击托盘图标切换显示状态
  - 气泡通知提示
  - 窗口关闭时最小化到托盘

- 🎨 **窗口功能**
  - 可拖动的悬浮窗口
  - 边缘自动吸附
  - 透明度可调（0.1 - 1.0）
  - 大小缩放支持
  - 始终置顶显示

- 📡 **WebSocket 服务器**
  - 本地 WebSocket 服务（localhost:19876）
  - 多客户端连接支持
  - 自动重连机制
  - 状态广播功能

- 🔌 **VSCode 扩展**
  - 文件轮询状态检测（500ms 间隔）
  - WebSocket 自动重连
  - 完整的命令面板支持
  - 状态变化日志输出

### 🔧 技术实现

- 基于 .NET 8.0 WPF 框架
- 自包含单文件发布（~170MB）
- 使用 System.Windows.Forms 实现托盘图标
- 动态 GDI+ 绘制托盘图标（无需外部资源文件）
- TypeScript 编写的 VSCode 扩展

### 📦 发布配置

- 完善的发布脚本 `publish.bat`
  - 自动清理旧构建
  - 版本号读取（从项目文件）
  - 单文件自包含发布
  - ReadyToRun 编译优化
  - 发布文件打包
  - 构建摘要和文件大小显示

## 版本说明

### 版本号格式

```
MAJOR.MINOR.PATCH
```

- **MAJOR（主版本号）**：不兼容的 API 变更
- **MINOR（次版本号）**：向后兼容的功能性新增
- **PATCH（修订号）**：向后兼容的问题修正

### 变更类型

- `Added` - 新功能
- `Changed` - 功能变更
- `Deprecated` - 即将废弃的功能
- `Removed` - 已移除的功能
- `Fixed` - Bug 修复
- `Security` - 安全相关修复
